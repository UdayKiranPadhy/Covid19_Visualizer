5 Chrome Extensions
Useful For Web Developers

1. ColorPick Eyedropper

This tool is so handy when you need to pick colors from a webpage. It comes with a zoomed eyedropper that allows easy color picking. The colour codes are available in RGB as well as in HEX.

2. WhatFont

I have been using this extension for a while now. It makes identifying fonts from any webpage so much easier. Along with the font family, it also provides additional details line font style, weight, size, etc.

3. Full Page Screen Capture

As the name suggests, it captures each part of a webpage as a single screenshot. You can then edit it or download it as PDF or Image.

4. Magic CSS

This is a live editor for CSS, Less, and Sass. With this extension, You can instantly add custom CSS styles to any website.

5. Dimensions

This is a perfect tool to measure the distance between elements on a website. It works pretty well with images too.
