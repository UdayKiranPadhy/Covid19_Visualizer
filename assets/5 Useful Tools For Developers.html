5 Tools & Websites
Useful For Web Developers

1. Matcherator

Ever looked at some text in an image and wondered what font it was? Just upload the image and it will suggest you a list of matching/ similar fonts.

2. Autoprefixer

You don't have to worry about adding vendor prefixes manually anymore. This tool adds, arranges, and deletes prefixes for you.

3. Coolors

I recommend this website a lot for a lot of different purposes. It allows you to generate a beautiful palette from your photos.

4. Remove.bg

Wanted to remove background from an awesome picture but had no knowledge about the fancy tools do to so? This tool allows you to remove the background from a picture in just 1 click.

5. SVGOMG

If you are looking for a tool for SVG optimization, then you should check this out. It has a pretty good number of options for configuration.
